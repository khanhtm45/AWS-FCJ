**2. Click on Model Catalog to find Claude Haiku 4.5**
![alt text](https://paperkite-master.github.io/AWS_FCJ/images/5-Workshop/5.3-Setup-Amazon-Bedrock/5.3.1-Check-model/image-1.png)

**3. Click on Claude Haiku 4.5 and scroll down to find the Model ID**
![alt text](https://paperkite-master.github.io/AWS_FCJ/images/5-Workshop/5.3-Setup-Amazon-Bedrock/5.3.1-Check-model/image-2.png)

**4. You can click "open in playground" to test the model**
![alt text](https://paperkite-master.github.io/AWS_FCJ/images/5-Workshop/5.3-Setup-Amazon-Bedrock/5.3.1-Check-model/image-3.png)

**Note: Currently you no longer need to go to model access to request model access. You can directly use the API of a model in Amazon Bedrock with an IAM account that has AWS marketplace permission once you call the model once to activate it**
