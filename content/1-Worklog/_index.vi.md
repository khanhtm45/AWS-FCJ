**Trong trang này** bạn sẽ tìm thấy nhật ký công việc toàn diện trong **12 tuần** ghi lại hành trình học AWS của tôi thông qua chương trình First Cloud Journey. Nhật ký này trải dài 3 tháng thực hành chuyên sâu với các dịch vụ AWS, từ các khái niệm cơ bản đến kiến trúc serverless nâng cao.

## Tổng quan Chương trình

Nhật ký này ghi lại quá trình học tập qua **12 tuần** với các workshop AWS, hoàn thành **60 bài thực hành** bao quát nhiều dịch vụ AWS và mô hình kiến trúc khác nhau. Mỗi tuần tập trung vào các chủ đề cụ thể, xây dựng kiến thức từng bước từ các dịch vụ nền tảng đến giải pháp cấp doanh nghiệp.

## Tiến độ Hàng tuần

**Tuần 1:** [Nền tảng AWS - Thiết lập Tài khoản, Ngân sách, Hỗ trợ, IAM và VPC](1.1-week1/)

- Tạo tài khoản AWS và cấu hình cảnh báo chi phí
- Khám phá các gói AWS Support và best practices
- Nắm vững quản lý người dùng và quyền IAM
- Xây dựng hạ tầng mạng VPC cơ bản

**Tuần 2:** [Compute và Storage Cốt lõi - EC2, IAM Roles, Cloud9, S3 và RDS](1.2-week2/)

- Triển khai và quản lý EC2 instances
- Triển khai IAM roles cho truy cập tài nguyên bảo mật
- Thiết lập môi trường phát triển Cloud9
- Cấu hình S3 buckets và RDS databases

**Tuần 3:** [Triển khai Ứng dụng - Lightsail, Containers, Auto Scaling, Monitoring và DNS](1.3-week3/)

- Khởi chạy ứng dụng trên Amazon Lightsail
- Làm việc với Docker containers và ECR
- Cấu hình Auto Scaling cho high availability
- Thiết lập CloudWatch monitoring và Route 53 DNS

**Tuần 4:** [CLI và Dịch vụ Database - AWS CLI, DynamoDB, ElastiCache, Networking và CDN](1.4-week4/)

- Nắm vững AWS CLI cho tự động hóa
- Xây dựng ứng dụng NoSQL với DynamoDB
- Triển khai caching với ElastiCache
- Cấu hình CloudFront cho content delivery

**Tuần 5:** [Migration và Disaster Recovery - VM Import/Export, Database Migration, DRS, Lambda và Grafana](1.5-week5/)

- Di chuyển VMs và databases lên AWS
- Triển khai giải pháp disaster recovery
- Tối ưu chi phí Lambda functions
- Thiết lập Grafana cho observability

**Tuần 6:** [Monitoring và Governance - CloudWatch, Resource Tags, IAM Policies và Systems Manager](1.6-week6/)

- Triển khai CloudWatch monitoring toàn diện
- Tổ chức tài nguyên với tags và groups
- Triển khai tag-based access control
- Tự động hóa operations với Systems Manager

**Tuần 7:** [Infrastructure as Code - CloudFormation, CDK và EC2 Optimization](1.7-week7/)

- Xây dựng infrastructure với CloudFormation templates
- Phát triển với AWS CDK sử dụng TypeScript/Python
- Tạo nested stacks và advanced patterns
- Tối ưu EC2 sizing với Compute Optimizer

**Tuần 8:** [Security và Automation - WAF, KMS, Secrets Manager, Cognito và Image Builder](1.8-week8/)

- Bảo vệ ứng dụng với AWS WAF
- Mã hóa dữ liệu với KMS
- Quản lý secrets an toàn
- Triển khai authentication với Cognito
- Tự động hóa patching với EC2 Image Builder

**Tuần 9:** [Networking và Messaging - AWS Backup, VPC Peering, Transit Gateway, SNS/SQS và EBS Multi-Attach](1.9-week9/)

- Tự động hóa backups cho business continuity
- Kết nối VPCs với Peering và Transit Gateway
- Xây dựng kiến trúc event-driven với SNS/SQS
- Chia sẻ dữ liệu an toàn với EBS NVMe reservations

**Tuần 10:** [CI/CD và Serverless - CodePipeline, Triển khai ECS/EKS, DynamoDB và Step Functions](1.10-week10/)

- Xây dựng CI/CD pipelines cho containers
- Triển khai lên ECS và EKS với automation
- Khám phá advanced DynamoDB patterns
- Điều phối workflows với Step Functions

**Tuần 11:** [Ứng dụng Serverless - CodeStar, Lambda Functions, API Gateway, Cognito và Messaging](1.11-week11/)

- Tự động hóa releases với AWS CodeStar
- Xây dựng serverless backends với Lambda
- Tạo RESTful APIs với API Gateway
- Triển khai user authentication
- Xử lý orders với SQS và SNS

**Tuần 12:** [Serverless Nâng cao - CI/CD, Document Management, Amplify, SSL/TLS và Monitoring](1.12-week12/)

- Tự động hóa serverless deployments
- Xây dựng hệ thống quản lý tài liệu
- Tích hợp Amplify cho auth và storage
- Bảo mật websites với SSL certificates
- Giám sát với CloudWatch và X-Ray

## Thành tựu Chính

- Hoàn thành **60 workshop AWS thực hành** trong 12 tuần
- Đạt được chuyên môn trong **25+ dịch vụ AWS**
- Xây dựng kiến trúc production-ready bao gồm serverless, containerized và ứng dụng truyền thống
- Nắm vững Infrastructure as Code với CloudFormation và CDK
- Triển khai security, monitoring và CI/CD practices cấp doanh nghiệp
- Sẵn sàng thiết kế và triển khai các giải pháp cloud có khả năng mở rộng
